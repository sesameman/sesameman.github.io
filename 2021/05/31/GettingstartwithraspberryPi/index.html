Getting start with raspberryPi

Getting start with raspberryPi

树莓派是基于Linux的单片机电脑，由英国树莓派基金会开发，目的是以低价硬件及自由软件促进学校的基本计算机科学教育。

树莓派的优势：

  1. 体积小巧，面积约为信用卡大小，高度约为4 cm；

  2. 高度可扩展（以树莓派4B为例）：有可编程针脚、有一个千兆网口（故而可以做软路由，NAS等）、支持单口4K60HZ输出，双口4K30HZ输出，type-c供电，两个usb3.0，两个usb2.0接口；

  3. 价格便宜，在官方提供的商城链接8GB-RAM版本只需要520元；

    ![截屏2021-05-15 上午10.23.14](/Users/kjy/Library/Application Support/typora-user-images/截屏2021-05-15 上午10.23.14.png)

  4. 用途广泛，以后不用了还可以做这些事情；

  5. 如果你选择装他自己做的系统==raspberryOS==(基于Debian)，系统会自带Mathematica及其内核，我觉得应该是免费的（就是给教育者的优惠一类的）。

树莓派的特点（不知道算是优点还是缺点）：

  1. 树莓派采用ARM（具体可以百度下）架构，优点也有缺点也有，我不好说；
  2. 只有一块儿芯片，所有显示器键盘鼠标都得额外购买（也可以写好Wi-Fi连接文件，直接用SSH连接，但在校园网环境中难度有点大），可以选购官方套件（带外壳显示器键盘一类的）。但最好我觉得就是从闲鱼上买一块儿，便宜好多；

树莓派的缺点：

  1. 性能不大行，我购买的raspberryPi-4B应该是树莓派性能天花板了，如果装了ubuntu图形界面则会卡顿（使用自己的raspberryos不卡）；

子祺自行斟酌吧！
